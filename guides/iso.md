1. Select `English` as language.

   ![iso-installation-step-1](assets/ISO_STEP_1.jpg)

2. Select `English (US)` keyboard layout and variant.

   ![iso-installation-step-2](assets/ISO_STEP_2.jpg)

3. Select `Ubuntu Server (minimized)` and enable third-party driver search if available.

   ![iso-installation-step-3](assets/ISO_STEP_3.jpg)

4. Leave network on **DHCP** (the install script reconfigures it later).

   ![iso-installation-step-4](assets/ISO_STEP_4.jpg)

5. Skip HTTP proxy.

   ![iso-installation-step-5](assets/ISO_STEP_5.jpg)

6. Keep the **default** APT mirror.

   ![iso-installation-step-6](assets/ISO_STEP_6.jpg)

7. Uncheck `Set up this disk as an LVM group`. Ensure `Use an entire disk` is selected.

   ![iso-installation-step-7](assets/ISO_STEP_7.jpg)

8. Continue with defaults.

   ![iso-installation-step-8](assets/ISO_STEP_8.jpg)

9. Create user `coolblock`, set hostname to `panel`, password to `coolblock` (changed later by end customer).

   ![iso-installation-step-9](assets/ISO_STEP_9.jpg)

10. Skip `Ubuntu Pro`.

    ![iso-installation-step-10](assets/ISO_STEP_10.jpg)

11. Select `Install OpenSSH server`.

    ![iso-installation-step-11](assets/ISO_STEP_11.jpg)

12. Do **not** select any additional snap packages.

    ![iso-installation-step-12](assets/ISO_STEP_12.jpg)

13. Reboot when installation completes.

    ![iso-installation-step-13](assets/ISO_STEP_13.jpg)
